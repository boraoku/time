## ūüöÄ Quick Start

### Prerequisites

- Ruby 3.4.5 or higher
- Rails 8.0 or higher
- Bundler

### Installation

```bash
# Clone the repository
git clone https://github.com/boraoku/time.git
cd time

# Install dependencies
bundle install

# Create and migrate database
rails db:create
rails db:migrate

# Start the server
rails server
```

Visit `http://localhost:3000` and start syncing times!
